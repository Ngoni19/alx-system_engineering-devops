* [Upstart documentation](http://upstart.ubuntu.com/cookbook/)
* [Application Server vs. Web Server](https://www.nginx.com/resources/glossary/application-server-vs-web-server/)


### Environment
* Technologies: Gunicorn, Upstart scripts
* OS: Ubuntu 14.04 LTS
* Web Server: Nginx
